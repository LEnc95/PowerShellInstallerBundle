
PowerShell Module Installer
===========================

This package installs essential Microsoft PowerShell modules for SharePoint, Exchange, Graph API, Identity, and Security.

Instructions:
-------------
1. Ensure both files are in the same folder:
   - PowerShellInstaller.ps1
   - LaunchInstaller.bat

2. Double-click 'LaunchInstaller.bat' to run the installer.
   - This will bypass execution policy restrictions.
   - A log file will be saved in your Documents folder.

3. Follow any prompts during installation.

Note:
-----
You may need to restart PowerShell after installation.
